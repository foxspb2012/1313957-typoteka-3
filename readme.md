# Как запустить проект

- `npm i`
- `npm run filldb 20`
- `npm start`

---

## Структура файла `.env`

- `DB_HOST` — домен подключения к БД
- `DB_PORT` — порт подключения к БД
- `DB_NAME` — наименование БД
- `DB_USER` — имя пользователя БД
- `DB_PASSWORD` — пароль пользователя БД
- `SESSION_SECRET`— строка секрета сеанса
